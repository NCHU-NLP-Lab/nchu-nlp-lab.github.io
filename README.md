# UDIC Website

實驗室的網站

## Structure
```bash
├── Dockerfile
├── README.md
├── package-lock.json
├── package.json # node dependencies & commands
├── requirements.txt # flask dependencies
├── server.js # run node server
├── server.py # run flask server
├── src # source code
└── webpack.config.js 
```

## Install

```bash
npm install
```
## Development  

```bash
npm run dev
```
## Compile for production

```bash
npm run build
```
